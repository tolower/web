3.编译运行调试
---
* 安装和配置go开发环境
* 安装liteide开发工具
* 安装mysql数据库（5.0版本，数据库引擎设置为innerDB）
* 安装beego环境和bee工具
* 用git客户端拉取项目开发代码
* 用bee工具运行代码
* 用liteide调试代码（当前打开为main.go文件，编译、运行、调试才正常）
